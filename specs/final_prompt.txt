# Project Specification

## User Requirement
build a backend for online bakery shop that sells cakes

## Selected Technology Stack

### 1. Web Development

### 2. Backend
**Why chosen:** Back-end is necessary to handle data, manage transactions, and provide the required functionality for an online bakery shop that sells cakes.
**Purpose:** Enables managing inventory, processing orders, handling user authentication, and facilitating communication between the frontend and database.

### 3. REST
**Why chosen:** REST is a popular and well-documented API standard that provides a simple and flexible way to communicate between the backend and frontend, which makes it suitable for building the backend for an online bakery shop.
**Purpose:** Enables efficient communication and resource handling

### 4. JavaScript/TypeScript
**Why chosen:** JavaScript/TypeScript is widely used for web development and has a rich ecosystem for backend services with Node.js.
**Purpose:** Enables the creation of a scalable, fast and flexible backend for an online bakery shop.

### 5. Express.js
**Why chosen:** Widely adopted and well-documented, it provides a fast and simple way to build APIs for our online bakery shop.
**Purpose:** Enables quick development of backend services

### 6. Caddy
**Why chosen:** It is a modern, easy-to-use web server with HTTPS support out of the box and automatic TLS.
**Purpose:** Enables serving the online bakery shop's website

## Technology Stack Summary
Web Development → Backend → REST → JavaScript/TypeScript → Express.js → Caddy
