# Project Specification

## User Requirement
build a backend for online bakery shop

## Selected Technology Stack

### 1. Web Development

### 2. Backend
**Why chosen:** To build a backend for online bakery shop
**Purpose:** Enables creation of server-side logic and API

### 3. REST
**Why chosen:** most widely adopted and easy to implement
**Purpose:** enables building a scalable and maintainable API

### 4. JavaScript/TypeScript
**Why chosen:** Modern and popular choice for building scalable and maintainable RESTful APIs
**Purpose:** Enables building a fast, secure, and efficient backend for the online bakery shop

### 5. NestJS
**Why chosen:** Robust, scalable, and maintainable framework with strong typing
**Purpose:** Enables building a robust and scalable backend for the online bakery shop

### 6. NGINX
**Why chosen:** NGINX is a popular and widely-used web server that can efficiently handle high traffic and is well-suited for a production environment.
**Purpose:** To serve as a reverse proxy and handle incoming HTTP requests

## Technology Stack Summary
Web Development → Backend → REST → JavaScript/TypeScript → NestJS → NGINX
