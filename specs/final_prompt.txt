# Project Specification

## User Requirement
build a backend for online bakery shop

## Selected Technology Stack

### 1. Web Development

### 2. Backend
**Why chosen:** The user requirement is to build a backend for an online bakery shop, which involves managing data and server operations.
**Purpose:** This option enables the development of the necessary serverside logic and database management for the online bakery application.

### 3. REST
**Why chosen:** REST is widely used and understood in the web development community, making it easier for developers to work on and maintain.
**Purpose:** REST enables the creation of stateless API endpoints to handle requests and responses between the backend and frontend.

### 4. Java
**Why chosen:** Java is a versatile, robust, and scalable language with a rich ecosystem of libraries for building web applications and APIs.
**Purpose:** enables development of the backend for the online bakery shop

### 5. Spring Boot
**Why chosen:** Popularity, extensive community support and rich ecosystem make Spring Boot an excellent choice for building the backend for an online bakery shop.
**Purpose:** Enables rapid development of production-ready spring-based applications

### 6. Jetty
**Why chosen:** Jetty is lightweight and easy to integrate with Spring Boot, providing a simple and efficient solution for running our application.
**Purpose:** Enables deployment of the online bakery shop backend application.

## Technology Stack Summary
Web Development → Backend → REST → Java → Spring Boot → Jetty
